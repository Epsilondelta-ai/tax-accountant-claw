# 2026년 근로소득 간이세액표

> 적용 기간: 2026년 2월 급여 지급분부터 (매년 조정)
> 근거법령: 소득세법 제129조, 소득세법 시행령 제194조
> 용도: 매월 급여 지급 시 원천징수할 소득세·주민세 결정

---

## 1. 간이세액표 개요

### 원천징수 의무

- 법인은 근로자에게 급여를 지급할 때 **소득세 + 주민세(소득세의 10%)**를 원천징수
- 매월 **다음 달 10일**까지 원천세 신고·납부
- 연말정산으로 최종 세액 정산 (매년 2월)

### 공제대상 가족 수 산정

- 본인 포함 (최소 1인)
- 배우자 (연 소득 100만원 이하)
- 부양가족 (직계존비속, 형제자매 등 요건 충족 시)
- 20세 이하 자녀 수에 따라 추가 공제

### 원천징수 세액 선택

근로자는 아래 3가지 중 선택 가능:

| 구분 | 비율 | 특징 |
|---|---|---|
| 80% 선택 | 간이세액의 80% | 연말정산 시 추가 납부 가능성 높음 |
| 100% 선택 | 간이세액의 100% | 기본 (대부분 선택) |
| 120% 선택 | 간이세액의 120% | 연말정산 시 환급 가능성 높음 |

## 3. 비과세 급여 항목

원천징수 시 아래 항목은 **과세대상 급여에서 제외**:

| 항목 | 비과세 한도 | 비고 |
|---|---|---|
| 식대 | 월 20만원 | 2023년~ 인상 |
| 자가운전보조금 | 월 20만원 | 본인 차량 업무용 사용 시 |
| 출산·보육수당 | 월 20만원 | 6세 이하 자녀 |
| 연구활동비 | 월 20만원 | 기업부설연구소 등 |
| 생산직 야간근로수당 | 월 240만원 한도 | 총급여 3,000만원 이하 |
| 국외근로소득 | 월 100~300만원 | 근무지역에 따라 차등 |

### 과세대상 급여 계산

```
과세대상 급여 = 총 급여 - 비과세 항목 합계
```

**예시**: 총 급여 320만원, 식대 20만원 별도 지급 시
→ 과세대상 급여 = 320만원 - 20만원 = **300만원** 구간 적용

## 4. 계산 공식 (간이세액표 산출 원리)

### 기본 구조

```
1. 연간 급여 추정 = 월급여 × 12
2. 근로소득공제 적용
3. 인적공제 적용 (기본공제 1인당 150만원)
4. 표준세액공제 적용
5. 산출세액 계산 (종합소득세율 적용)
6. 근로소득세액공제 적용
7. 월 원천징수세액 = 연 산출세액 ÷ 12
```

## 5. 월급 300만원 원천징수 예시 (본인 + 배우자 = 2인)

### 전제

- 과세대상 월급여: 300만원 (비과세 제외 후)
- 공제대상 가족 수: 2인 (본인 + 배우자)

### 원천징수 세액

```
소득세: 47,600원
주민세: 47,600 × 10% = 4,760원
원천징수 합계: 52,360원
```

### 4대보험까지 포함한 실수령액 계산

```

- 국민연금: 142,500원

- 장기요양: 7,086원
- 고용보험: 27,000원
- 소득세: 47,600원
- 주민세: 4,760원
────────────────────
공제 합계: 336,796원
실수령액: 2,663,204원
```

## 6. 원천세 신고·납부 안내

### 신고 주기

| 구분 | 신고 주기 | 기한 |
|---|---|---|
| 일반 사업장 | 매월 | 다음 달 10일 |
| 반기별 납부 (상시고용 20인 이하) | 반기 | 1월 10일, 7월 10일 |

> **5인 미만 스타트업**: 반기별 납부 신청 가능 (관할 세무서에 신청)

### 신고 방법

- **홈택스**: 원천징수이행상황신고서 전자신고
- **위택스**: 지방소득세(주민세) 신고·납부
- **납부서**: 소득세(국세)와 지방소득세(지방세) 각각 납부

### 연말정산

- **시기**: 매년 2월 급여 지급 시
- **방법**: 근로자 소득·세액공제 자료 수집 → 정산 → 추가 납부 또는 환급
- **간소화서비스**: 국세청 홈택스에서 1월 중순 오픈
