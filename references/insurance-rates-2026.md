## 2. 국민연금

### 기본 정보

- **보험료율**: 9.5% (2026년 인상)
- **부담 비율**: 근로자 4.75% + 사업주 4.75%
- **기준소득월액 상한**: 617만원 (2026년 7월 변경 예정)
- **기준소득월액 하한**: 39만원

### 요율 변경 이력

| 연도 | 보험료율 | 비고 |
|---|---|---|
| ~2025년 | 9.0% | 근로자 4.5% + 사업주 4.5% |
| 2026년~ | 9.5% | 근로자 4.75% + 사업주 4.75% |

> **참고**: 국민연금 재정안정화를 위해 단계적 인상 진행 중. 향후 추가 인상 예정.

### 적용 제외 대상

- 18세 미만, 60세 이상 근로자
- 월 60시간 미만 단시간 근로자 (3개월 이상 근무 시 적용)
- 국민연금법 제8조에 따른 적용제외 사유 해당자

## 3. 건강보험

### 기본 정보

- **보험료율**: 7.19% (보수월액 기준)
- **부담 비율**: 근로자 3.595% + 사업주 3.595%
- **보수월액 상한**: 약 119,625,106원/월
- **보수월액 하한**: 약 279,266원/월

### 장기요양보험

- **요율**: 건강보험료의 13.14%
- **부담 비율**: 근로자 50% + 사업주 50%
- 건강보험료에 추가 부과되는 형태

### 건강보험 연말정산

- 매년 4월, 전년도 보수총액 기준으로 정산
- 차액 추가 납부 또는 환급 발생

## 6. 월급 300만원 기준 계산 예시

### 전제 조건

- 월 보수: 3,000,000원
- 업종: 소프트웨어 개발업 (산재보험 0.7%)
- 사업장 규모: 150인 미만

### 근로자 부담분

| 보험 종류 | 계산 | 금액 |
|---|---|---|
| 국민연금 | 3,000,000 × 4.75% | 142,500원 |
| 건강보험 | 3,000,000 × 3.595% | 107,850원 |
| 장기요양보험 | 107,850 × 13.14% ÷ 2 | 7,086원 |
| 고용보험 (실업급여) | 3,000,000 × 0.9% | 27,000원 |
| **근로자 합계** | | **284,436원** |

### 사업주 부담분

| 보험 종류 | 계산 | 금액 |
|---|---|---|
| 국민연금 | 3,000,000 × 4.75% | 142,500원 |
| 건강보험 | 3,000,000 × 3.595% | 107,850원 |
| 장기요양보험 | 107,850 × 13.14% ÷ 2 | 7,086원 |
| 고용보험 (실업급여) | 3,000,000 × 0.9% | 27,000원 |
| 고용보험 (고용안정) | 3,000,000 × 0.25% | 7,500원 |
| 산재보험 | 3,000,000 × 0.7% | 21,000원 |
| **사업주 합계** | | **312,936원** |

### 요약

| 구분 | 금액 |
|---|---|
| 월급 (세전) | 3,000,000원 |

| **사업주 실질 인건비** | **3,312,936원** |

> **참고**: 실수령액은 여기서 소득세·주민세 원천징수액을 추가로 차감해야 함
