# 부가가치세 신고 규칙

> 근거법령: 부가가치세법
> 대상: 법인사업자 (일반과세자)
> 법인사업자는 간이과세 적용 불가

---

## 1. 부가가치세 기본 구조

### 세율

| 구분 | 세율 | 적용 대상 |
|---|---|---|
| 일반 과세 | **10%** | 대부분의 재화·용역 |
| 영세율 | **0%** | 수출, 외화획득 등 |
| 면세 | **비과세** | 기초생활필수품, 의료, 교육 등 |

### 기본 계산 공식

```
납부세액 = 매출세액 - 매입세액
매출세액 = 과세표준(공급가액) × 10%
매입세액 = 세금계산서상 매입세액 합계
```

- **납부세액 > 0**: 세금 납부
- **납부세액 < 0**: 환급 (매입세액이 더 큰 경우)

---

## 2. 신고 기간 및 기한

### 법인사업자 신고 일정 (연 4회)

| 신고 구분 | 과세 기간 | 대상 기간 | 신고·납부 기한 |
|---|---|---|---|
| 1기 예정 | 1월 1일 ~ 3월 31일 | 1~3월 | **4월 25일** |
| 1기 확정 | 1월 1일 ~ 6월 30일 | 4~6월 (1~6월 정산) | **7월 25일** |
| 2기 예정 | 7월 1일 ~ 9월 30일 | 7~9월 | **10월 25일** |
| 2기 확정 | 7월 1일 ~ 12월 31일 | 10~12월 (7~12월 정산) | **다음해 1월 25일** |

> **참고**: 기한일이 공휴일·토요일인 경우 다음 영업일로 연장

### 예정신고 vs 확정신고

- **예정신고**: 해당 분기 3개월분 신고
- **확정신고**: 반기(6개월) 전체 정산 (예정신고 납부세액 차감)

### 예정고지

- 직전 과세기간 납부세액의 50%를 고지서로 발부
- 법인사업자는 예정고지 대신 **예정신고** 의무 (원칙)
- 다만, 소규모 법인(직전기 공급가액 1.5억 미만)은 예정고지 가능

---

## 3. 영세율 적용 대상

### 영세율이란?

- 세율 0%를 적용하여 매출세액은 0원이지만, **매입세액은 전액 환급** 가능
- 수출기업에 유리한 제도

### 영세율 적용 대상 (부가가치세법 제21~24조)

| 구분 | 내용 |
|---|---|
| 재화의 수출 | 직접 수출, 중계무역, 위탁판매 수출 등 |
| 용역의 국외공급 | 국외에서 제공하는 용역 |
| 외화획득 재화·용역 | 비거주자·외국법인에 공급하는 재화·용역 (대가를 외화로 수령) |
| 선박·항공기의 외국항행용역 | |
| 외국인 전용 관광기념품 판매 | |
| 외교관 면세 | 주한 외교관에 대한 공급 |

### 영세율 적용 시 제출 서류

- 수출실적명세서
- 수출신고필증 (관세청)
- 외화입금증명서 (은행)
- 영세율매출명세서

> **IT 스타트업**: 해외 고객에게 SaaS, 개발 용역 제공 시 영세율 적용 가능.
> 단, 외화 수령 증빙(외화입금증명서) 반드시 확보 필요.

---

## 4. 면세 대상

### 면세란?

- 부가가치세가 아예 부과되지 않음
- 매출세액 없음, **매입세액 공제도 불가**

### 주요 면세 대상 (부가가치세법 제26조)

| 구분 | 세부 내용 |
|---|---|
| 기초생활필수품 | 미가공 농·축·수·임산물, 수돗물, 연탄·무연탄 |
| 의료·건강 | 의료보건용역 (병·의원), 혈액 |
| 교육 | 학교 교육용역, 교과서 |
| 여객운송 | 시내버스, 지하철 등 (항공·택시 제외) |
| 도서·신문·잡지 | 도서, 신문, 잡지 (광고 제외) |
| 금융·보험 | 금융·보험 용역 |
| 토지 | 토지의 공급 (건물 제외) |
| 인적용역 | 작가·예술가·강사 등의 인적용역 |
| 주택임대 | 국민주택규모 이하 주택임대 |

## 5. 공통매입세액 안분계산

### 언제 필요한가?

과세사업과 면세사업을 **겸영**하는 경우, 공통으로 사용하는 매입세액을 안분

### 계산 공식

```
공제 가능 매입세액 = 공통매입세액 × (과세 공급가액 ÷ 총 공급가액)
```

### 계산 예시

```
총 매출: 10억원 (과세 8억 + 면세 2억)
공통매입세액: 500만원

공제 가능 = 500만원 × (8억 ÷ 10억) = 400만원
불공제 = 500만원 - 400만원 = 100만원
```

### 정산

- 예정신고 시: 직전 과세기간 비율로 안분 (예정)
- 확정신고 시: 해당 과세기간 실제 비율로 정산

## 6. 매입세액 불공제 항목

다음 항목은 세금계산서를 수취했더라도 **매입세액 공제 불가**:

| 항목 | 설명 |
|---|---|
| 비영업용 소형승용차 | 1,000cc 초과 승용차 구입·유지비 (운수업 등 제외) |
| 접대비 관련 | 접대비로 지출한 재화·용역의 매입세액 |
| 사업과 무관한 지출 | 업무무관 자산 취득·유지비 |
| 면세사업 관련 | 면세사업에만 사용하는 매입세액 |
| 토지 관련 | 토지 취득·조성 관련 매입세액 |
| 세금계산서 미수취 | 적격증빙 없는 거래 |
| 사업자등록 전 매입 | 사업자등록 신청일로부터 역산 20일 전 이전 매입 |

## 7. 세금계산서

### 전자세금계산서 의무발행

- **법인사업자**: 전자세금계산서 **의무** 발행
- 발행 기한: 공급시기가 속하는 달의 **다음 달 10일**까지
- 지연발행 가산세: 공급가액의 1%
- 미발행 가산세: 공급가액의 2%

### 세금계산서 필수 기재사항

1. 공급자의 등록번호·성명(법인명)
2. 공급받는 자의 등록번호
3. 공급가액·세액
4. 작성 연월일

### 수정세금계산서

발행 사유:
- 환입 (반품)
- 계약의 해제
- 공급가액 변동
- 내국신용장 사후 개설
- 착오에 의한 이중발행 등

## 8. 신용카드 매출 세액공제

### 공제 대상

- **연 매출(공급가액) 10억원 이하** 개인·법인 사업자

### 공제율 및 한도

| 구분 | 공제율 | 연간 한도 |
|---|---|---|
| 신용카드·직불카드 매출 | 1.3% | 1,000만원 |

> **주의**: 법인사업자 중 연 매출 10억원 초과 시 공제 불가

### 공제 요건

- 신용카드 매출전표 또는 현금영수증 발행분
- 전자세금계산서와 중복 발행 시 공제 불가

---

## 9. 가산세

### 주요 가산세 항목

| 위반 유형 | 가산세율 |
|---|---|
| 무신고 | 납부세액의 20% (부정: 40%) |
| 과소신고 | 과소신고 납부세액의 10% (부정: 40%) |
| 납부지연 | 미납세액 × 0.022% × 경과일수 |
| 세금계산서 미발행 | 공급가액의 2% |
| 세금계산서 지연발행 | 공급가액의 1% |
| 세금계산서 지연수취 | 공급가액의 0.5% |
| 매출처별 세금계산서합계표 미제출 | 공급가액의 0.5% |
| 전자세금계산서 미전송 | 공급가액의 0.3% |
| 현금영수증 미발행 (의무발행 업종) | 미발행 금액의 20% |

## 10. 부가세 신고 체크리스트 (법인용)

### 신고 전 확인사항

- [ ] 전자세금계산서 발행·수취 내역 확인 (홈택스)
- [ ] 신용카드 매출·매입 내역 조회
- [ ] 현금영수증 발행·수취 내역 조회
- [ ] 영세율 매출 증빙서류 준비 (해당 시)
- [ ] 불공제 매입세액 구분
- [ ] 공통매입세액 안분계산 (겸영사업자)
- [ ] 고정자산 매입 내역 확인 (감가상각 자산)
- [ ] 대손세액공제 해당 여부 확인

### 제출 서류

1. 부가가치세 신고서
2. 매출처별 세금계산서합계표
3. 매입처별 세금계산서합계표
4. 신용카드 매출전표 등 수령명세서
5. 영세율 첨부서류 (해당 시)
6. 대손세액공제 신고서 (해당 시)

## 11. 스타트업 실무 팁

### 매입세액 공제 최대화

- 사업 관련 모든 지출은 **세금계산서** 또는 **신용카드/현금영수증**으로 증빙
- 사업자 명의 법인카드 사용 권장
- 개인카드 사용 시에도 사업자번호로 현금영수증 발행 요청

### 조기환급

- 영세율 매출이 있는 경우: 예정신고 시 조기환급 신청 가능
- 시설투자 시: 감가상각자산 취득 관련 매입세액 조기환급 가능
- 조기환급 기한: 신고기한 후 15일 이내 환급

### 수출 IT 서비스

- 해외 클라이언트에 SaaS, 개발 용역 제공 → **영세율 적용**
- 외화입금증명서 보관 필수
- AWS, Google Cloud 등 해외 서비스 이용료: 대리납부 의무 (부가세법 제52조)
